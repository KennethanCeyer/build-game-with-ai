# 3D Agent Playtest Lab Solution

This folder contains the completed ADK wiring for the 3D game demo.

For the hands-on, compare this file with the starter version:

- solution: `solution/agentic_game_demo/agent_setup.py`
- starter: `starter/agentic_game_demo/agent_setup.py`

The rest of the 3D game implementation stays in the main package under
`src/agentic_game_demo`. Participants only need to understand the Agent, Tool,
and LoopAgent wiring in this small file.
