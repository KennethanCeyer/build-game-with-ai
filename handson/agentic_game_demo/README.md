# 3D Agent Playtest Lab Starter

Edit only `starter/agentic_game_demo/agent_setup.py`.

The game runtime, MCP tool implementation, UI, and tests are already provided by
the main package. The exercise is to fill the small ADK wiring gaps:

1. choose the Gemini model,
2. name the ADK controller agent,
3. connect the MCP server module,
4. attach MCP tools and `exit_loop`,
5. wrap the controller in a `LoopAgent`.

When stuck, compare against `solution/agentic_game_demo/agent_setup.py`.
